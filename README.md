DSA Foundations Checklist

A minimal web-based checklist to track **core Data Structures & Algorithms practice** for beginners.

Live Site: [https://yogalakshmi-sekar.github.io/dsa-checklist/](https://yogalakshmi-sekar.github.io/dsa-checklist/)

What is this?

A simple productivity tool to help students **build strong DSA foundations** through structured problem tracking.

Topics Covered

* Basic Programming
* Strings
* Patterns
* Arrays (Easy–Medium)
* Lists
* Stacks
* Queues
* Trees (Easy)

Tech Used

**HTML • CSS • GitHub Pages**

Who is this for?

Students & beginners preparing for coding tests and interviews.
